# ImageConverter
This is to convert the jpg file to text file.  Using the ASCII value to create the image as plain text.


---------------------------------------------------------
 - File Conversion -
A tool to create the text file from the image.jpg.
Default output file name will be same as input file name, and by default color will not be revers.

```
ImageConverter /i Image.jpg [/o ImageFileAsText.txt] [/r 0 or 1] [/h 50%] [/w 100%]

/i      -       Fully specified input image file path.
/o      -       outout text file path.
/r      -       0(default) OR 1. 1 reverse the color.[Black-White]
/h      -       Specify the Height 0-100%
/w      -       Specify the Width 0-100%

eg.
        1.      ImageConverter /i Image.jpg /o ImageFileAsText.txt
                it will generate the output ImageFileAsText.txt
        2.      ImageConverter /i Image.jpg
                it will generate the output Image.txt
        3.      ImageConverter /i Image.jpg /r 1
                it will generate the output Image.txt, color reverse
```
